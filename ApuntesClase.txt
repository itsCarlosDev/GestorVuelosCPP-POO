

// Recursividad: lo que hace la Recursividad es separar la solucion en varias soluciones mas simples

Variables → camelCase, empiezan minúscula: nomAlumne.
Funciones → verbos en camelCase: calculaMitja().
Clases/Tipos → Mayúscula inicial: class Alumne.
Constantes → MAYÚSCULAS con _: NOMBRE_MAXIM.
Atributos privados → m_ delante: m_nomAlumne.
Variables globales → ::variable.

En punteros → char* nom, no char *nom.

Funciones
Si recibe objetos → por referencia (const& si no los modifica).

Bucles
NO usar goto, continue, break (salvo en switch).

Clases
- Orden → public primero, luego private.
- Siempre un constructor por defecto.
- Inicialización de atributos → lista en constructor:
MyClass(int v) : m_value(v) {}
